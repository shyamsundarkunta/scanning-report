# Network Reconnaissance Project

## Project Summary

This project demonstrates network reconnaissance techniques using Nmap against the authorized TryHackMe Nmap room. The objective was to identify live hosts, discover open ports, enumerate services, perform OS fingerprinting, compare scan types, and evaluate basic firewall evasion techniques while documenting all findings in a professional reconnaissance report.
